O Zingen é uma landing page moderna para uma plataforma de Karaokê, desenvolvida com foco total em experiência do usuário (UX) e performance. Este projeto foi construído aplicando conceitos avançados de design responsivo e arquitetura de componentes CSS.

📱 Diferenciais do Projeto
Mobile-First: A estrutura foi pensada primeiramente para dispositivos móveis, garantindo uma navegação fluida em telas menores antes da expansão para Desktop.

Design Moderno: Uso de gradientes, tipografia otimizada e elementos visuais que remetem ao universo musical e tecnológico.

Acessibilidade: Tags semânticas de HTML5 para melhor leitura por tecnologias assistivas e SEO.

Performance: Imagens otimizadas e código limpo para carregamento instantâneo.

🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando o "core" do desenvolvimento web moderno:

HTML5: Estrutura semântica e robusta.

CSS3: Estilização avançada com Flexbox e CSS Grid.

Google Fonts: Integração de tipografia personalizada para melhor identidade visual.

Metodologias: Organização de código focada em escalabilidade e manutenção.
